User Flows

Onboarding

New user signs up, completes profile setup, and reaches their dashboard for the first time.

Landing → Sign Up → Profile Setup → Dashboard

Entry point: User clicks "Get Started" on landing page

Key decisions: Email vs social signup, optional profile fields

Success state: User lands on personalized dashboard

Edge cases: Email already exists, social auth fails

Core Task

User navigates to the main feature, completes the primary action, and sees confirmation.

Dashboard → Browse → Detail → Confirmation

Entry point: User clicks primary CTA on dashboard

Key decisions: Filter/sort options, item selection

Success state: Action completed, confirmation shown

Edge cases: Empty states, error handling
